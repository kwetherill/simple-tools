Login Manager

A lightweight Chrome extension to store and manage your site credentials — right in your browser toolbar.

Download Extension
Version 1.0.0

All your logins in one place

Store URLs alongside their usernames and passwords in a clean, scrollable list — no more hunting through notes or sticky pads.

Stays local

Credentials are stored in Chrome's local extension storage — they never leave your browser and require no account or server.

Add & remove instantly

One click to add a new entry, one click to remove it. No forms with ten fields — just the URL and credentials you need.

How to Install

1

Download the zip

Click the Download Extension button above to save Login Manager Plugin 1.0.0.zip to your computer.

2

Unzip the file

Extract the zip to a folder you'll keep on your machine (e.g. ~/Extensions/logins-manager).

3

Open Chrome Extensions

In Chrome, navigate to chrome://extensions and enable Developer mode using the toggle in the top-right corner.

4

Load the extension

Click Load unpacked and select the folder you extracted in step 2.

5

Pin & use

Click the puzzle-piece icon in Chrome's toolbar, find Login Manager, and pin it. You're all set!
